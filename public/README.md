## Avatar

Para adicionar sua foto de perfil:
1. Coloque a imagem nesta pasta (ex: `avatar.jpg` ou `avatar.png`)
2. Atualize o campo `avatarUrl` no arquivo `src/data/config.json`:
   - Se a imagem estiver aqui: `avatarUrl: "/avatar.jpg"`
   - Se for uma URL externa: `avatarUrl: "https://exemplo.com/avatar.jpg"`
